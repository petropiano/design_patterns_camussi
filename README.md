# PORTFÓLIO DE PADRÕES DE PROJETO (DESIGN PATTERNS)
Este repositório contém as implementações práticas de três Padrões de Projeto clássicos (GoF), desenvolvidos como parte da disciplina de **Engenharia de Software**.

**Aluno:** Otavio Camussi Tomaz
**RA:** 2707314

Este repositório foi organizado em três diretórios, contemplando um padrão de cada categoria:

 - **Builder** (Criacional);
  - **Facade** (Estrutural);
  -  **Strategy** (Comportamental);
  
  Assim, cada pasta contém:
  
 - Um código em PYTHON (.py) que representa uma aplicação prática do padrão de
   projeto; 
 - Um README contedo:
	 - a definição do padrão;
	 - o problema que soluciona;
	 - como soluciona; 
	 - um diagrama UML explicado 
	 - o código  comentado bloco a bloco;



Todo o embasamento teórico, diagramas UML e conceitos utilizados para a elaboração deste código foram baseados no material do [**Refactoring Guru**](https://refactoring.guru/).
